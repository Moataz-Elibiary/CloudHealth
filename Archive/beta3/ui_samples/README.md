# CloudHealth Beta 3 UI Mockups

These files are static HTML/CSS representations of the CloudHealth Beta 3 dashboard. They allow you to review the UI design and report layouts while offline.

### Available Samples:
- **success.html**: A complete run where all clusters passed or had minor warnings. Use this to test report expansion.
- **failed.html**: A scenario with critical failures, showing how error details are presented.
- **running.html**: A simulation of an active diagnostic session with glowing status pills.

### Key Features to Review:
1.  **Glassmorphism Theme**: Translucent cards with backdrop filters and vibrant gradients.
2.  **Headline Zone**: Real-time status area (blue border) that shows what the backend is doing.
3.  **Expandable Reports**: Click on a section header (e.g., "etcd Health") to see the detailed checks inside.
4.  **Status Pills**: Responsive status indicators (PASS, FAIL, WARN, RUNNING) with distinct color coding.
